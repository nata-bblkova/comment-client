Клиент для осуществления запросов к серверу с комментариями
=====================

установить библиотеку можно с помощью команды:
-----------------------------------
composer require nata/comment-client

требуемые зависимости: 
-----------------------------------
php: >=8.2
ext-curl: *
php-макет /php-mock-phpunit: ^2.10
phpunit/phpunit: ^11
